Omnigraph

Omnigraph is a single-page web application for visualizing the positive/ negative sentiment of Instagram posts across the US. Accesses the Instagram API for the 30 largest US cities, and for any other city entered as a search query. Uses npm-sentiment to evaluate text, and displays the data with D3.
